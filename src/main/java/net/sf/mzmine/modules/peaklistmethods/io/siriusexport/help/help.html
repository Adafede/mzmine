Sirius Export

Description

This module exports fragmentation spectra of peaks in the peak list into MGF (Mascot Generic Format) format file. This module is specifically formatted for use with the Sirius web-platform. Sirius can be downloaded at the following address. Sirius results can be mapped into GNPS molecular networks. See the documentation here. In addition to the spectra, the following peak characteristics are saved into the file:

FEATURE_ID
Row number in the peak list
PEPMASS
Mass of the peak. If each row in the peak list contains several peaks (e.g. after performing alignment), the mass of the highest peak is chosen
RTINSECONDS
Retention time of the peak. If each row in the peak list contains several peaks, the average retention time is chosen
SCANS
The same as FEATURE_ID
CHARGE
The charge for the specified row
MSLEVEL
Either equals 1 (MS) or 2 (MS/MS)
