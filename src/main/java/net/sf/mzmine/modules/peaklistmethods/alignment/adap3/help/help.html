ADAP Aligner

Description

This method aligns detected compounds in different samples through matching their mass fragmentation spectra and retention times.

First, all compounds within a retention time range are collected from all samples. Then, pair-wise scores between those compounds are calculated as follows:

S(C1, C2) = w Stime + (1 - w) Sspec
where Stime is the relative retention time difference between two compounds and Sspec is the spectrum similarity between two compounds. Finally, a two-phase procedure is used to determine the best matching compound in each sample. For more details, see the paper

Jiang, W.; Qiu, Y.; Ni, Y.; Su, M.; Jia, W.; Du, X.: An automated data analysis pipeline for GC-TOF-MS metabonomics studies. Journal of proteome research 2010, 9 (11), 5974-81

Parameters

  • Min confidence: smallest fraction of samples where a compound should be detected. If the fraction of samples where a compound is detected is larger then Min confidence, then that compound is included in the list of aligned compounds. Otherwise, that compound is discarded. This parameter should be between 0 and 1.
  • Retention time range: maximum allowed retention time gap between aligned compounds
  • Score tolerance: the minimum matching score required for compounds to be included in the list of aligned compounds. This parameter should be between 0 and 1
  • Score weight: the weight w that is used in the scoring formula. If w = 0, then only the spectrum similarity is used for scoring. If w = 1, then only the retention time proximity is used for scoring.
  • EIC score: method used for calculating the retention time proximity. The retention time difference (fast) is preferred method.
