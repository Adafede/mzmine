MSn feature list builder

Description

The MSn feature list builder detects MS1 precursor signals which possess an MSn scan. The modules scans the raw data looking for MSn scans and builds a chromatographic feature using the specified m/z and retention time ranges for each precursor ion with a corresponding MSn scan.

If more than one peak is detected in the range or more precise peak integration is required, the feature list may be deconvoluted via the Chromatogram Deconvolution tool.

Method parameters

MS level
The MS level for precursor feature building. Do not set MS Level in the Scan Selection field!
m/z Tolerance
Precursor ion (MS1) mass tolerance for feature building.
Retention time tolerance
Precursor ion (MS1) retention time tolerance for feature building.
