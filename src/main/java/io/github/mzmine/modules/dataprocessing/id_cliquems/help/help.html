CliqueMS

Description

This module uses the CliqueMS algorithm [1] to group and annotate the features.

Parameters Description

  • PeakList - Feature List on which cliqueMS algorithm is to be performed.
  • Filter - Filter out similar features if they pass all MZ, RT and intensity tolerances. Two features are considered similar if their relative change in m/z, rt and intensity are less than the respective tolerances.
  • MZ tolerance - If Filter option is set, this value determines the similarity of two features, if their relative difference of m/z values is less than this, they are considered similar.
  • RT tolerance - If Filter option is set, this value determines the similarity of two features, if their relative difference of retention times values is less than this, they are considered similar.
  • Intensity tolerance - If Filter option is set, this value determines the similarity of two features, if their relative difference of intensity is less than this, they are considered similar.
  • Log - likelihood tolerance - The Log likelihood function is maximised for clique formation. The iterations are stopped when the relative change in current log likelihood with respect to the initial log likelihood is less than the log-likelihood tolerance value.
