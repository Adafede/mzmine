Mobilogram interpolation

This module can be used to interpolate between zero-values in mobilograms, if enough non-zero values are detected in the neighbourhood. This can be used to interpolate in for high resolution IMS data in the case that no ion package arrives at the detector due to low ion current. The use of this module is experimental, since it can also lead to a lot of noise being detected as a feature in later processing steps. (see figure 1, figure 2.)

Therefore, this module should only be used for targeted approaches only when a specific compound has to be detected and instrumental optimisations cannot achieve the required result. For non-target approaches, the use of this module is strongly discouraged!

Parameters

Window width The number of consecutive mobility (and their corresponding intensity) values to be check by this algorithm.
Number of non-zero intensities The number of non-zero intensities in the Window width to be eligible for interpolation.
Interpolation filter width The maximum number of values (=mobility scans) to be checked for neighbouring non-zero intensities during interpolation. If no value is found in that window, it will be interpolated from 0.
