3D plot

Description

This tool presents a three dimensional plot where X axis represents the retention time, Y axis the m/z value and Z axis the intensity of the signal. This plot is the collection of all the information from the raw data in a graphical representation.

Parameters

MS level
This refers to the scan level (MS1,MS2,... ,MSn) to be used to display the chromatogram.
Retention time
Retention time (X axis) range.
Retention time resolution
Number of data points on retention time axis.
m/z range
Range of m/z values.
m/z resolution
Number of data points on m/z axis.

Additionally in the parameters window, the button "Set automatically" allows the user to set all the ranges automatically. MZmine will use the maximum ranges allowed by the raw data.

Functionality

This plot is using the third part library JfreeChart for its basic functionality. To zoom in and out, use the third button functionality of the mouse. Also by click and dragging mouse's left button, the user can "rotate" the plot in any direction. If right button is used instead, the user can "move" the plot to any position.
