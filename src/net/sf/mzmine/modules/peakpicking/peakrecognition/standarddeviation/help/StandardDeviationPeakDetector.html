Standard deviation peak recognition

Chromatogram analysis

This peak recognizer sets the level to use as a baseline using the standard deviation (in terms of intensity) of the signal. An algorithm calculates the baseline level following the next steps:

1.- Calculate the standard deviation.
2.- Get the differences (absolute values) between the standard deviation and any point (mass spectrum peak) with an intensity below of the standard deviation value.
3.- Get the Quantile (the percent of points below the given value) value of all differences. The Quantile is calculated according with "Standard Deviation threshold level" parameter.
4.- The baseline is the standard deviation less the Quantile of differences.


Parameters

This recognizer uses a three parameters:

"Min peak height"

This value sets the minimum acceptable height (intensity) for a chromatographic peak.

"Min peak duration"

This value sets the minimum acceptable length (time duration) for a chromatographic peak.

"Standard Deviation threshold level"

This value is the Quantile of differences between the standard deviation and all points below of standard deviation.
