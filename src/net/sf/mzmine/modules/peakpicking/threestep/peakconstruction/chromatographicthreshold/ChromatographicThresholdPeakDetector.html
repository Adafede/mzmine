Centroid Mass Detector

Spectrum analysis

This mass detector search for each data point in the current spectrum, due each data used to represent mass spectral peaks, is in terms of two parameters, the centroid (the weighted center of mass) and the intensity. Every one of this data point is considered as part of a possible peak.

Parameters

This detector use a single parameter "noise level". This value sets the minimum intensity level for a centroided data point must have to be consider as part of a possible peak.
