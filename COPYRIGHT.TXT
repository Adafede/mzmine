Information about used third-party libraries

	MZmine uses some pieces of third-party code. We are thankful to their authors 
	for the code. These libraries are included with this source code packet 
	(in directory lib/) to make compiling of MZmine easier.
	
	NetCDF Java Library by the Unidata community is available from 
	http://my.unidata.ucar.edu/content/software/netcdf-java/index.html under 
	GNU Lesser General Public License (LGPL).
	
	Danby Package for probability and statistics by Charles S. Stanton available from
	http://www.math.csusb.edu/faculty/stanton/danby/danby_package.html

	ExampleFileFilter.java, TableMap.java, TableSorter.java by Sun Microsystems, Inc. 
	are available in JDK's demos. For license and copyright information, see below.
	
	Base64.java by Robert Harder is available from http://iharder.net/xmlizable and 
	is placed in the Public Domain.
	
	JRAP by SASHIMI-project is available from http://sashimi.sourceforge.net under 
	GNU General Public License (GPL).
	
	Xstream is used to serialize objects to XML and available 
	from xstream.codehaus.org under BSD license. 
	
	Xpp3 is used to deserialize object from XML and available 
	from http://www.extreme.indiana.edu/xgws/xsoap/xpp/mxp1/index.html
