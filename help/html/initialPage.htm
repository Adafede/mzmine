MZmine 2 - LC/MS ToolBox
   

Key features

The MZmine toolbox (v. 2.0) contains:

  • GUI for import, visualization, processing, and export of spectra.

  • Smoothing and filtering of spectra.

  • Peak detection for continuous and centroid spectral data.

  • Spectral alignment.

  • Normalization methods.

  • Compatibility with NetCDF, mzXML (v2.0,v 2.1), mzData(v1.05) and new mzML (v1.0) data formats.

  • Possibility to distribute data processing on multi-processor computers and computing clusters.

  • Batch processing.

  • Improved visualization and processing of centroided data.

  • New filtering methods.

  • Isotope pattern detection.

  • Database connectivity for peak identification.

  • Methods for processing high-resolution instrument data.

  • Method for retention time normalization.

  • Support for MS^n scans.

  • 3D and 2D visualizer and implementations of Chromatogram, Spectra.

  • PCA plot for visualizing results.

  • Storage and retrieval of project (opened datafiles, generated peak lists, etc.).

  • Project parameters for defining properties of samples.

  • Help.
